# MLOps Course - AMMI

This repository contains the materials for the MLOps course at AMMI. Each folder is structured by the day of the course and contains the corresponding lecture slides and lab materials.

## Repository Structure

The repository is organized by the day of the lesson. Each day has its own folder containing lecture slides and a lab folder with all relevant materials for the hands-on exercises.

## Usage

1. Navigate to the relevant day's folder (day1, day2, ... day9).
2. Review the lecture slides pdf for the concepts covered during that session.
3. Open the lab folder for hands-on exercises:
    - Go through the lab slides pfd for an overview of the session.
    - Attempt the tasks outlined in the markdown file. The file also contains the solutions.
    - Check the code folder for any provided code necessary for the tasks.
